Question-2: What is the difference between innerHTML, innerText, and textContent ?
1.innerHTML is used to get and set html content of an element.
2.InnerText is used to get and set inside text of an element.
3.textContent is used to get and set all text content in an element incuding hidden text.
Question-3: What is event delegation in the DOM?
Event delegation is a technique where a single event listener is added to a parent element instead of multiple child elements.
Question-4: What is event bubbling in the DOM?
Event bubbling is a technique in the DOM where an event starts from the target element and move up to its parent elements until it reaches the root
